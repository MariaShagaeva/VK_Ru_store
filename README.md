# VK_Ru_store
MVP-версия магазина приложений RuStore для Android
# Демонстрация работы: 

**Выполненные требования**

**Обязательные:**

Онбординг - показывается только один раз при первом запуске приложения

Витрина приложений - список всех приложений с иконкой, названием, кратким описанием и категорией

Карточка приложения - детальная информация с полным описанием, скриншотами и возрастным рейтингом

** Опциональные:**

Список категорий приложений - навигация по иконке категорий в правом верхнем углу

Полноэкранный просмотр скриншотов - по клику на скриншот в карточке приложения
