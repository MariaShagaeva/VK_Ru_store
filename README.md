# VK_Ru_store
MVP-версия магазина приложений RuStore для Android

**Демонстрация работы:** https://youtu.be/nk2sNZI12zo

Видео показывает: запуск приложения, онбординг, список приложений, карточку приложения, скриншоты, категории

**Выполненные требования**

**Обязательные:**

- Онбординг - показывается только один раз при первом запуске приложения
- Витрина приложений - список всех приложений с иконкой, названием, кратким описанием и категорией
- Карточка приложения - детальная информация с полным описанием, скриншотами и возрастным рейтингом

**Опциональные:**

- Список категорий приложений - навигация по иконке категорий в правом верхнем углу
- Полноэкранный просмотр скриншотов - по клику на скриншот в карточке приложения
- Работа с API (не выполнено - локальные данные)

**Технологии**

- Kotlin, Jetpack Compose
- MVVM архитектура
- Navigation Component
- Минимальная SDK: Android 7.0

**Как запустить**

**1. Клонирование репозитория**

git clone https://github.com/MariaShagaeva/VK_Ru_store.git

**2. Открытие в Android Studio**

1. Запустите Android Studio
2. Выберите File → Open
3. Найдите и откройте папку VK_Ru_store

**3. Сборка и запуск**

1. Подключите Android-устройство или запустите эмулятор
2. Нажмите зелёную кнопку Run ▶️
3. Приложение будет установлено и запущено автоматически

**Минимальные требования:** Android 7.0 (API 24) или выше
